# Oracle & PostgreSQL SQL Practice

This repository contains SQL and PL/SQL practice examples for Oracle and PostgreSQL databases.
It focuses on real-world queries, performance optimization and backend database concepts.

## Topics Covered
- Joins & Subqueries
- Functions & Procedures
- Indexing & Performance Tuning
- Common SQL Interview Queries

## Technologies
- Oracle SQL / PL/SQL
- PostgreSQL SQL / PL/SQL

## Purpose
To demonstrate hands-on SQL and database optimization skills for backend developer roles.
